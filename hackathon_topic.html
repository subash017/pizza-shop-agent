Hackathon Round 1

Build a Real-World OpenEnv Environment

Create a complete environment that an AI agent can learn from using the standard step() / reset() / state() API, with deterministic grading, meaningful rewards, and deployment-ready packaging.

Hackathon Topic: Pizza Shop Operations Simulator with OpenEnv

Core Requirements

  • Simulate a real-world workflow (not a game).
  • Typed models for Observation, Action, and Reward.
  • At least 3 graded tasks (easy, medium, hard).
  • Reward shaping with partial progress signals.

Must-Have Endpoints

  • /tasks for task list + action schema.
  • /grader for post-episode scoring.
  • /baseline for reproducible model run.

Submission Readiness

  • Working Dockerfile and local run flow.
  • HF Space deployment compatibility.
  • README with setup, spaces, and baseline scores.
  • Validation-friendly OpenEnv project structure.

Tip: Keep graders deterministic, scores in the range [0.0, 1.0], and ensure baseline behavior is reproducible.
